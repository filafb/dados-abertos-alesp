# To run locally:
`git clone git@github.com:filafb/dados-abertos-alesp.git`

`npm run install`

Create a DB named `dados-abertos`
  - using createdb utility:
    `createdb dados-abertos`
  - using `psql`
    `create database dados-abertos`

Then:
- `npm run start-dev`
